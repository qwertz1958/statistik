# Darstellung unternehmenskritischer Daten in Echtzeit
+ Darstellungsformen:
	+ Text
	+ Tabelle
	+ Diagramme
	+ Kartenmaterial
+ Vorteile:
	+ kostengünstig
	+ geringer personeller Aufwand
	+ Daten verlassen nicht die Firma
	+ Abbildung im Intranet der Firma
